STEP 1 — INSTALL NINJATRADER (FREE)

Download NinjaTrader 8 from ninjatrader.com — free for simulation. FTMO and many prop firms support NinjaTrader.

STEP 2 — ENABLE MARKET DATA BROADCAST

In NinjaTrader: Tools → Options → Market Data → Enable Web API

This opens a local server on your PC that this engine can read from:

http://localhost:8080/api/marketdata/ES 03-25
⚠ Engine and NinjaTrader must be on same computer for localhost to work
STEP 3 — CLICK "NINJATRADER LOCAL" IN ENGINE

Once NinjaTrader is running with ES chart open, click the NinjaTrader button in the data bar. The engine polls your local NinjaTrader every second for live prices.
